🔗 DNSSEC — Seguridad de DNS

Firma criptográfica de registros DNS para prevenir envenenamiento de cache.

Configuración

DNSSEC_ENABLED=true

Cadena de Confianza

  Root Key (KSK) ──▶ TLD (.com/.org) ──▶ Zone Key (ZSK) ──▶ crux.local
       │                    │                     │                 │
       │ 2048-bit RSA       │ 2048-bit RSA        │ 256-bit ECDSA   │
       │ Chain of Trust     │ DS Record           │ RRSIG           │
       │                    │                     │                 │
       └── Validation path ─┴─────────────────────┴─────────────────┘
  • KSK (Key Signing Key): Firma la ZSK. Rotación anual
  • ZSK (Zone Signing Key): Firma los registros A/AAAA/MX/TXT. Rotación mensual
  • RRSIG: Firma del registro individual, con TTL limitado

Impacto en Autenticación de Correo

DNSSEC asegura que las consultas de DKIM, SPF y DMARC no sean manipuladas:

Registro Firma DNSSEC Efecto sin DNSSEC Efecto con DNSSEC
DKIM Sí Clave pública puede ser falsificada Integridad de clave verificada
SPF Sí Lista blanca puede ser modificada Lista blanca intacta
DMARC Sí Policy puede ser alterada Policy verificada

🛡️ Hardening con Seccomp y eBPF

Restricción de syscalls y monitorización kernel-level con eBPF.

Seccomp Profile

Cada contenedor ejecuta con un perfil seccomp estricto que limita las syscalls permitidas al mínimo necesario para la operación del servicio:


// infra/security/seccomp-profile.json
{
  "defaultAction": "SCMP_ACT_ERRNO",
  "defaultErrnoRet": 1,
  "archMap": [
    {
      "architecture": "SCMP_ARCH_X86_64",
      "architectureCompat": true,
      "enabled": true
    }
  ],
  "syscalls": [
    {
      "names": ["accept", "bind", "connect", "epoll_wait", ...],
      "action": "SCMP_ACT_ALLOW"
    }
  ]
}
Efecto: Si un proceso intenta ejecutar una syscall no listada (ej: mount, ptrace, reboot), el kernel devuelve EACCES inmediatamente — sin crashear el contenedor.

Capacidades de Linux (Capabilities)

Principio de mínimo privilegio aplicado a Linux capabilities. Todos los contenederos ejecutan con cap_drop: ALL y solo añaden lo estrictamente necesario:

Contenedor Capabilities añadidas Justificación
Nginx NET_BIND_SERVICE Puerto 443 (< 1024)
Postfix NET_BIND, SETGID, SETUID, CHOWN, DAC_OVERRIDE, FOWNER, SYS_CHROOT MTA con chroot y manejo de archivos
Postgres DAC_OVERRIDE, FOWNER, SETGID, SETUID, CHOWN, NET_BIND DB engine con file ownership
Redis DAC_OVERRIDE Acceso a datos RDB/AOF
Node Exporter NET_RAW SNMP / ICMP metrics

Otras Hardening Measures

  • read_only: true: Filesystem de root en read-only. Solo tmpfs para datos volátiles
  • no-new-privileges: true: Imposibilita elevación de privilegios (SUID/SGID bypass)
  • user: 1000:1000: Ejecución como usuario no-root en todos los contenedores de app
  • tmpfs limits: Limites estrictos de memoria para /tmp, /var/run, /var/cache

eBPF (Avanzado)

eBPF (extended Berkeley Packet Filter) permite monitorizar y filtrar tráfico de red a nivel kernel sin overhead. En Crux-Webmail se utiliza para:
  • Traffic filtering: Reglas dinámicas en el datapath de cada contenedor
  • Anomaly detection: Detección de patrones anómalos en syscalls
  • Observability: Telemetría de bajo nivel para Prometheus/Grafana

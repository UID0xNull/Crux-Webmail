🌐 Servicio JMAP

Protocolo JMAP RFC 8620 para operaciones CRUD de email.

¿Qué es JMAP?

JMAP (JSON Meta Application Protocol, RFC 8620) es un protocolo moderno diseñado para reemplazar IMAP, SMTP y CalDAV. Usa HTTP POST con JSON y permite batching, filtrado del lado servidor y sync eficiente.

Implementación del Servicio

// JMAP router handler
app.post('/jmap', {
  schema: {
    body: z.object({
      using: z.array(z.string()),
      methodCalls: z.array(z.tuple([
        z.string(),         // method name
        z.object({}),       // arguments
        z.null(),          // invocation data
      ])),
    }),
  },
  handler: async (req, reply) => {
    const { methodCalls } = req.body;
    const responses = await Promise.all(
      methodCalls.map(async ([method, args]) => {
        const handler = jmapMethods.get(method);
        if (!handler) return [method, { error: 'method_not_found' }, null];
        const result = await handler(args, req.user);
        return [method, result, null];
      })
    );
    return { methodResponses: responses };
  },
});

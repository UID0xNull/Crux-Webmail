Capas de Seguridad

┌──────────────────────────────────────────────┐
│  Capa 1: Transport (TLS 1.3 / mTLS)          │
│  └── Nginx SSL termination, OCSP stapling     │
├──────────────────────────────────────────────┤
│  Capa 2: Network (Microsegmentación Docker)  │
│  └── 4 redes aisladas, no interconectadas    │
├──────────────────────────────────────────────┤
│  Capa 3: Application (Auth + Validation)     │
│  └── JWT, Zod schemas, rate limiting          │
├──────────────────────────────────────────────┤
│  Capa 4: Content (WASM Sanitization + CSP)   │
│  └── DOMPurify WASM, strict CSP headers       │
├──────────────────────────────────────────────┤
│  Capa 5: Data (Encryption at rest + in flight)│
│  └── WebCrypto E2E, AES-256-GCM, TLS 1.3     │
├──────────────────────────────────────────────┤
│  Capa 6: Kernel (eBPF seccomp + AppArmor)    │
│  └── Syscall whitelist, capability drop ALL   │
└──────────────────────────────────────────────┘

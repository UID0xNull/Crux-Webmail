Versión actual: 1.0.0 — Backend Fastify 5 + Frontend Next.js 15 operativos. Infraestructura Zero-Trust completa con 18 servicios Docker. IMAP/SMTP/JMAP integrados. Suite de tests backend + frontend. Fases v0.1–v0.6 cerradas; v1.0.0 estable lanzado.

v0.1.0 — Fundación Zero-Trust (✅ Completado)

  • ✅ Infraestructura completa: Postfix + Dovecot + Amavis + ClamAV + MinIO en Docker
  • ✅ docker-compose.yml con 18 servicios, 4 redes aisladas (DMZ, App-tier, Data-tier, Monitor-tier), 12 volumes
  • ✅ docker-compose.prod.yml — configuración de producción con healthchecks estrictos y restart policies
  • ✅ Hardening Docker: read-only filesystems, no-new-privileges, cap_drop ALL, seccomp profiles
  • ✅ Microsegmentación de red interna (internal: true en redes de app y data)
  • ✅ mTLS con CA interna — script infra/scripts/mtls-setup.sh
  • ✅ TLS 1.3 obligatorio en todos los servicios (Nginx, Dovecot, Redis)
  • ✅ Secrets via Docker secrets (filesystem-mounted, no env vars en prod)
  • ✅ Monitoring stack completo: Prometheus + Grafana + Loki + Alertmanager + OTel Collector
  • ✅ Nginx reverse proxy con CSP estricto, HSTS, rate-limiting

v0.3.0 — Backend y Frontend Core (✅ Completado)

Backend — Fastify 5

  • ✅ Módulo Auth: JWT (fastify-jwt v9), refresh tokens, sesiones con Redis, 2FA TOTP (speakeasy), bcryptjs
  • ✅ Módulo IMAP Bridge: connection-manager con pools, adapter pattern, JMAP client service
  • ✅ Módulo SMTP: Nodemailer, envío con cifrado OpenPGP (openpgp v6.1.0)
  • ✅ Módulo Email: email queue (BullMQ), controlador REST, búsqueda con indexer
  • ✅ Módulo Mail: MIME pipeline completo — parser, attachment validator, ClamAV scanner, HTML sanitizer
  • ✅ Módulo WebSockets: ws-gateway, ws-bridge, ws-handler para notificaciones push en tiempo real
  • ✅ Servicios: attachment-service, draft-service
  • ✅ Middleware: auth, correlation-id, CORS estricto, CSRF, rate-limiter, security-headers (Helmet), Prometheus metrics
  • ✅ Observabilidad: OpenTelemetry SDK (trace + metrics exporters OTLP HTTP), Prometheus collector
  • ✅ ORM: Sequelize + PostgreSQL, caché con ioredis (Redis)

Frontend — Next.js 15 + React 19

  • ✅ Next.js 15 App Router con TypeScript estricto
  • ✅ Página Login: autenticación con 2FA TOTP
  • ✅ Dashboard: inbox con paginación, vista de detalle de mensajes, compose con adjuntos
  • ✅ Integración JMAP para CRUD de email
  • ✅ WebSockets para notificaciones en tiempo real
  • ✅ Cifrado E2E con WebCrypto + OpenPGP
  • ✅ SSR + ISR para SEO y carga rápida
  • ✅ Dynamic imports y tree-shaking para optimización de bundle
  • ✅ ErrorBoundary + PerformanceProvider integrados en DashboardLayout
  • ✅ PWA offline-first con IndexedDB (completado)

💻 Setup de Desarrollo Local

Requisitos, instalación y primeros pasos para contribuir.

Requisitos

  • Node.js ≥ 20 LTS
  • Docker ≥ 24 + Docker Compose
  • Git ≥ 2.40
  • RAM: Mínimo 8GB recomendado (16GB ideal)
  • Disk: ~5GB para imágenes Docker + volumes

Instalación Rápida

# 1. Clonar repo
git clone https://github.com/user/Crux-Webmail.git
cd Crux-Webmail

# 2. Instalar dependencias
npm install

# 3. Configurar variables
cp .env.example .env
# Editar .env con valores de desarrollo

# 4. Subir infraestructura
docker compose up -d postgres redis minio clamav dovecot postfix amavis

# 5. Ejecutar migraciones DB
npm run db:migrate

# 6. Seed datos de prueba
npm run db:seed

# 7. Iniciar dev server con hot reload
npm run dev

Estructura del Proyecto

Crux-Webmail/
├── src/
│   ├── server/           # Fastify backend
│   │   ├── config/       # Zod schemas, config helpers
│   │   ├── modules/      # IMAP, SMTP, JMAP bridges
│   │   ├── services/     # Bussiness logic
│   │   ├── middleware/   # Auth, CORS, rate limiting
│   │   ├── routes/       # API endpoints
│   │   └── models/       # Sequelize ORM
│   ├── frontend/         # Next.js web client
│   │   ├── app/          # App Router
│   │   ├── components/   # React componentes
│   │   └── lib/          # Utils compartidas
│   └── shared/           # Tipos TypeScript compartidos
├── infra/
│   ├── docker/           # Dockerfiles
│   ├── compose/          # docker-compose.yml
│   ├── security/         # Seccomp, AppArmor
│   └── nginx/            # Config Nginx
├── wiki/                 # Esta documentación
├── tests/                # Unit + E2E
├── scripts/              # Utilidades de deploy/build
└── docs/                 # Diagramas y specs

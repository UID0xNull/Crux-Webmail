📝 Guía de Código & Estilo

Convenciones, patterns y buenas prácticas.

Estilo General

  • Indentación: 2 espacios (TabWidth=2)
  • Max line length: 100 chars
  • Semi-colons: Yes (strict)
  • Quotes: Single quotes 'hello' (except JSX/JSX-like)
  • Trailing commas: Yes (ES5)
  • Arrow functions: Prefer over function expressions
  • const/let: No var, prefer const

// ✅ CORRECTO - Zod runtime validation
import { z } from 'zod';
const UserSchema = z.object({
  id: z.string().uuid(),
  email: z.string().email(),
  roles: z.array(z.string()).min(1),
});
type User = z.infer;

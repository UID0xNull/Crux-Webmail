🛡️ Amavis (Filtrado Antivirus & Anti-Spam)

Amavis como content filter entre Postfix y Dovecot, integrando ClamAV y SpamAssassin.

Arquitectura de Filtrado

┌────────────┐    SMTP:10024    ┌──────────┐    SMTP:10025    ┌──────────┐
│  Postfix  │ ────────────────► │  Amavis  │ ────────────────► │  Postfix  │
│  (sender) │                   │  (filter)│                   │  (local)  │
└───────────┘                   └────┬─────┘                   └───────────┘
                                     │                        ┌──────────┐
                          ┌──────────▼───────┐               │  Dovecot  │
                          │                  │               │  (LMTP)   │
                          │   ClamAV         │               └───────────┘
                          │   (antivirus)    │
                          │                  │
                          │   SpamAssassin   │
                          │   (anti-spam)    │
                          └──────────────────┘

Configuración en Docker

amavis:
  image: malthe/adagios-mta
  container_name: crux-amavis
  networks:
    services-network:
  ports:
    - "10024:10024"
    - "10025:10025"
  volumes:
    - amavis_data:/var/lib/amavis
    - clamav_db:/var/lib/clamav
  environment:
    - AMAVIS_DEBUG=0
    - CONTENT_FILTER_MODE=clamav,spamassassin

Reglas de Spam

AcciónScoreDescripción
Accept< 3.0Email limpio, pasa directo
Tag3.0 – 5.0Sospechoso: agrega headers X-Spam
Quarantine5.0 – 12.0Movido a carpeta Quarantine
Reject> 12.0Email rechazado (bounce al remitente)

Integración ClamAV

# ClamAV se conecta vía socket Unix
cd /var/lib/clamav/

# Actualizar DB de virus
freshclam

# Verificar versión de DB
clamscan --version

# Quarantine path
/var/lib/amavis/quarantine/

White/Black Listing

# Amavis @whitelist
# amavisd.conf snippet
@whitelist_sender_maps = (new_RE( {
    '.trusted-domain.com' => 1,
    'safe-user@example.org' => 1,
}));

# @blacklist
@blacklist_sender_maps = (new_RE( {
    '\.malicious-domain\.com$' => 1,
}));

Monitoreo

# Ver estadísticas de filtrado
amavisd stats

# Ver emails en quarantine
docker exec crux-amavis ls /var/lib/amavis/quarantine/

# Prometheus exporter
curl http://localhost:9090/api/v1/query?query=amavis_emails_scanned

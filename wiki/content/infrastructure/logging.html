📝 Logging & Telemetría

Logging estructurado JSON, tracing distribuido y retención escalable.

Pino Logger

// Config de Pino en el backend Fastify
const logger = pino({
  level: 'info',          // trace|debug|info|warn|error|fatal
  formatters: {
    level: (label) => ({ level: label.toUpperCase() }),
  },
  serializers: {
    req: serializers.req,
    res: serializers.res,
    err: serializers.err,
  },
  transport: {
    target: 'pino-loki',   // Envío directo a Loki
    options: { url: 'http://loki:3100/loki/api/v1/push' }
  }
})

Retención de Logs

StorageRetenciónMedio
Hot30 díasLoxB en /logs_shared
Warm90 díasMinIO bucket (parquet)
Cold365 díasS3 Glacier / MinIO archive

Categorías de Logs

  • App logs: Fastify + Pino — request/response lifecycle
  • Mail logs: Postfix/Dovecot — delivery, bounce, rejection
  • Security logs: Auth events, failed logins, rate limit hits
  • System logs: Container health, Docker events, network
  • Audit logs: Todas las operaciones de escritura (crear, modificar, eliminar)

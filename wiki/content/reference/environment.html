📋 Variables de Entorno Completas

Referencia completa de todas las variables de configuración del sistema.

Uso

# 1. Copiar template
cp .env.example .env

# 2. Personalizar valores

# 3. Iniciar servicios
npm run docker:up
Nota: Todos los valores de .env.example pasan por validación estricta de Zod al inicio del servidor. Si una variable es inválida, el servidor se detiene con error detallado.
